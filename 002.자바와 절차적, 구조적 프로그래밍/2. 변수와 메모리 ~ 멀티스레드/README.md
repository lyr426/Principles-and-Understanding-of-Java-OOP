### 학습목표 
- 변수들이 어떻게 메모리 상에서 동작하는지 알 수 있다. 
- 지역변수와 전역변수에 대해 알 수 있다. 
- 멀티스레드와 멀티프로세스에 대해 알 수 있다. 

### 키워드
지역변수, 전역변수, 클래스 멤버 변수, 객체 멤버 변수, static, 멀티스레드, 멀티프로세스
